---

## About the Project
Developed as part of the ERC-funded project **ChinaComx: Comics Culture in the PRC**, this repository hosts the digital tables of contents for:
* 《连环画研究》 *Lianhuanhua Research* (1955–1957, 1978–1983)
* 《连环画论丛》 *Lianhuanhua Discussion Series* (1983–1986)
* 《连环画艺术》 *Lianhuanhua Art* (1987–1991)

These journals offer critical insights into the internal discussions, political impacts, and reader feedback that shaped the Chinese socialist mediascape over several decades.
